Pendu est un jeu de pendu très basique écrit en Java avec LibGDX.

Il peut être déployé comme application java ou comme app Android.

Il est distribué sous license GPLv3 ce qui veut dire qu'il est libre et le restera.
Vous pouvez vous inspirer librement de ce travail, mais s'agissant d'un premier programme
en Java, il n'est pas très bien écrit et vous trouverez sans doute mieux écrit ailleurs !

Caractéristiques :

 * Tous les mots sont en majuscule sans accents en évitant les noms propres
 * Différents niveaux de jeu avec ou sans temps limite
 * Possibilité de choisir parmi plusieurs listes de mots
 * Sauvegarde des meilleurs scores
 * Affichage du nombre d'essais restant, du temps et du nombre de mots devinnés jusque là
 * S'adapte à la résolution (et l'orentation) de l'écran/la fenêtre
